## BreakageNotifier

### Stack:
* Python
  * aiohttp
  * SQLAlchemy
  * asyncio
  * pytelebot
* JavaScript
  * Vue.js
* Docker
* Rabbitmq
* Redis
* Telegram API
* Bash - Wrote script to update server in one action

### Responsibilities:
All tasks from design to deploy.

### Description:
It was a project for notifying repairmen about breakdowns at the factory and
monitoring equipment downtime. Workers marked some machines as broken,
the repairmen received a notification about it, fixed problems and filled out 
reports. All reports were collected in the admin panel, where they can be
sorted or downloaded.
